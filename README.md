# music-theory-python

```python
import chords

print(chords.degrees('E9'))         # ['1', '3', '5', '7', '9']
print(chords.degrees('Fmaj#11'))    # ['1', '3', '5', 'Î”', '9', '#11']
print(chords.midi_chord('Fmaj#11'))  # [65, 69, 72, 76, 79, 83]
```
